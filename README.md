# EduAI Chatbot

## Objetivo

Desarrollar un tutor virtual inteligente integrado a Moodle mediante una API REST utilizando Python y Flask.

## Tecnologías

- Python
- Flask
- Git
- GitHub
- Render
- Moodle
- Postman
- Inteligencia Artificial

## Integrantes

- Azul Natividad

## Descripción

EduAI Chatbot es un prototipo de tutor virtual inteligente que se integrará con Moodle para apoyar a los estudiantes mediante una API REST.
